# 🚀 StadiumFlow — Smart Stadium Companion System

![StadiumFlow](img/stadium_bg.png)

## 🌍 The Problem

Attending large-scale events should be exciting…  
but the reality is:

- ⏳ 20–40 minute queues at entry gates  
- 🚶 Unpredictable crowd congestion  
- ❌ No real-time navigation or guidance  

Current solutions focus on **tickets**, not the **actual experience inside the stadium**.

---

## 💡 The Solution

**StadiumFlow** transforms stadiums into **intelligent environments** that guide users in real-time.

👉 Instead of reacting to crowds, users are **proactively guided**.

---

## ⚙️ Core Features

### 👤 Fan Experience

- 🧭 **AI Fast Entry Plan**  
  Get the fastest gate based on real-time crowd density  

- 🍔 **Smart Food Recommendations**  
  Avoid long queues with AI-suggested alternatives  

- 🗺️ **Live Dynamic Routing**  
  Get rerouted instantly to avoid congestion  

- ⏱️ **Digital Queue System**  
  Wait virtually instead of standing in line  

---

### 🛠️ Staff Command Center

- 📊 **Live Crowd Monitoring**  
  Track density across zones  

- 🚪 **Gate Control System**  
  Open/close gates dynamically  

- 🚨 **Emergency Broadcast System**  
  Send alerts and manage evacuation flows  

---

## 🧠 System Intelligence

StadiumFlow continuously:

- Analyzes crowd density  
- Predicts waiting times  
- Optimizes routing decisions  

👉 Creating a **real-time feedback loop between users and infrastructure**

---

## 🏗️ Tech Stack (Google-Powered)

- 🔥 **Firebase** → Real-time data + Authentication  
- 🤖 **Gemini AI** → Smart recommendations  
- 🗺️ **Google Maps Platform** → Routing + visualization  
- ⚡ **Vanilla JS (ES Modules)** → Lightweight, fast execution  

---

## 🎬 Demo Flow

1. Enter as **Fan**
2. Get **AI gate recommendation**
3. Open **Map → Smart Route**
4. Check **Queues**
5. Switch to **Admin**
6. Control gates → See system react
